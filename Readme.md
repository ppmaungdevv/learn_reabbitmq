# Node app with Rabbitmq

#### Rabbitmq Learning Project

## Download & Installation

```
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.12-management
```

## Install rabbitmq package for node

```
npm install amqplib
```
